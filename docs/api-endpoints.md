# API Endpoints

## HTML API

### Root
- `GET /` - loads React web app

## JSON API

### Users
- `POST /api/users`
- `PATCH /api/users` 

### User Tickets
- `GET /api/users/:id/tickets`

### Session
- `POST /api/session`
- `DELETE /api/session`

### Events
- `GET /api/events`
- `POST /api/events`
- `GET /api/events/:id`
- `PATCH /api/events/:id`
- `DELETE /api/events/:id`

### Categories
- `GET /api/categories`
- `GET /api/categories/:id`
